# Sudoku Solver


## Build
```bash
chmod +x build
./build
```

## Note
- before run testcases gunzip the **data/puzzles_and_solutions.txt.gz** file.


## How to
- first build
- Run testcases 
    ```bash
    ./bin/test
    ```

- Run Program
